Setting up Environment(Platform.IO)¶

  1. First we need to install Microsoft Visual Studio Code. Follow the steps to download and install it.

    1. Go to https://code.visualstudio.com .

    2. Click on “Download” button at the top right corner of the screen.

    3. You will be directed to a new page. Choose your OS there and click on it.

    4. You then will be directed to documentation page of VS code and your download will begin.

NOTE(for beginners): Read the complete documentation page as it will help you to get comfortable with UI of VS code.

  1. Go to your downloaded file directory and install the file.

Congrats you have completed the very step.

  1. Now we can setup Platform.IO. Follow these steps to do it.

    1. Open VS code and go to extension manager.

    2. Search for PlatformIO IDE extension.

    3. Click on Install.
